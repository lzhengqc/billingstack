Install OpenJDK 1.6+

$sudo apt-get install openjdk-7-jdk

Install Grails 2.1.1

$sudo add-apt-repository ppa:groovy-dev/grails
$sudo apt-get update
$sudo apt-get -y install grails-ppa
$sudo apt-get install grails-2.1.2

Clone BillingStack Repository

$git clone http://github.com/stacksherpa/billingstack

Start the REST API

$cd billingstack
$grails run-app
